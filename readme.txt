== Description ==

Spacerify: A responsive spacer block with customizable values for desktop and mobile layouts.

== Installation ==

1. Download the plugin zip file or install the plugin through the WordPress plugins screen directly.
2. Go to the WordPress dashboard and navigate to Plugins > Add New.
3. Click the "Upload Plugin" button and select the zip file you just downloaded.
4. Click the "Install Now" button and activate the plugin.
5. The block will now be available in the editor under the "Layout" category.

== Frequently Asked Questions ==

Q: How can I set the spacing for desktop and mobile views using Spacerify?

A: Setting spacing for desktop and mobile views is a breeze with Spacerify! Simply add the block to your Gutenberg editor and input the desired height values for both desktop and mobile. Spacerify will automatically adapt the spacing to ensure a seamless experience across different devices.

Q: Can I customize the colors of the spacer block?

A: Yes, you can! Spacerify allows you to add colors to the spacer block. You have the flexibility to choose custom colors or utilize the colors from your theme's palette, ensuring a consistent and visually appealing design.
